## Getting Started
  
### Installation
  
1. Clone this repos (CIC)
```sh
git clone ***************************
```
2. Go To Folder Repo
```sh
code .
```
3. Install Module
```sh
npm install
```
  
4. Start website
```sh
npm run serve
```
### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
